# Shortcuts

This directory contains exported shortcuts for the Webcam Lamp Controller.

## Exporting Shortcuts

macOS does not allow command-line export of shortcuts. To export manually:

1. Open **Shortcuts.app**
2. Right-click each shortcut
3. Select **Share** â†’ **Export...**
4. Save to this directory

## Required Shortcuts

- **MeetingSOON** - Warning triggered 5 minutes before a meeting
- **MeetingON** - Triggered when webcam becomes active
- **MeetingOFF** - Triggered when webcam becomes inactive

## Importing Shortcuts

Double-click any `.shortcut` file to import it into Shortcuts.app.
